## Procedimiento para ejecutar
### 1. Crear el entorno virtual
```
python -m venv venv-Procesamiento_de_Imagenes_I
```
### 2. Activar el entorno virtual
#### Windows (git bash):
```
source venv-Procesamiento_de_Imagenes_I/Scripts/activate
```
#### Windows (cmd):
```
venv-Procesamiento_de_Imagenes_I/Scripts/activate.bat
```
#### Linux/MacOS
```
source venv-Procesamiento_de_Imagenes_I/bin/activate
```

### 3. Instalar dependencias
```
pip install -r requirements.txt

```
### 4. Para ver la resolución del ejercicio 1: Ecualización local de histograma
#### ejecutar el archivo:
```
Resolucion_tp1_ecualizacion.py
```
### 5. Para ver la resolución del ejercicio 2: Corrección de múltiple choice
#### ejecutar el archivo:
```
Resolucion_tp1_examenes.py
```

### Aclaración:
El archivo *Exploracion_tp1_examenes.py* contiene fracciones del código que se utilizaron como pruebas para llegar a la resolución del ejercicio 2; no es necesario ejecutar.
